# Project Bootstrap

Use this skill at project start.

## Procedure

1. Define scope and success metrics.
2. Create and activate `.venv`.
3. Confirm Python 3.12+.
4. Initialize `requirements.txt` and baseline dependencies.
5. Create standard layout and naming conventions.
6. Initialize Git and branch strategy.
7. Scaffold quality gates (lint, type-check, tests, security checks).
8. Create baseline docs (README, architecture notes, contributing, runbook).

## Output format

- Bootstrap checklist
- Proposed project structure
- Initial dependency set
- Initial quality gates
- First milestone plan
- Risks and assumptions
