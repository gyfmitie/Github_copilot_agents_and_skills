# Performance and Profiling

## Procedure

1. Define measurable performance targets.
2. Profile before optimizing.
3. Rank hotspots by impact.
4. Apply minimal-risk optimizations.
5. Re-measure and compare against baseline.

## Output format

- Target metrics
- Profiling plan
- Hotspot ranking
- Optimization candidates with tradeoffs
- Before/after validation plan
