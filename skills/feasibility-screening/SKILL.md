# Feasibility Screening

Use this skill when a new tool idea is proposed.

## Inputs to collect

- Use case and decision to improve
- Users and workflow context
- Available data sources and sampling rates
- Constraints: budget, timeline, compliance, infrastructure

## Procedure

1. Define success metric and decision impact
2. Check data sufficiency and leakage risk
3. Check engineering plausibility and constraints
4. Estimate implementation complexity
5. Identify top failure modes
6. Produce go/no-go recommendation with rationale

## Output format

- Feasibility score: 0 to 100
- Data readiness score: 0 to 100
- Value potential score: 0 to 100
- Top 5 blockers
- Mitigation actions
- Recommendation: Go, Go with conditions, or No-go
- 2-week proof-of-concept plan
